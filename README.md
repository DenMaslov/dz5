### INSTALLATION:
1. git clone https://github.com/DenMaslov/dz5.git
2. cd dz5
3. cd files_utility


for using:
  py main.py --operation move --src D:\\b...\\ --to D:\\a...\\ --threads 10
  py main.py --operation copy --src D:\\b...\\ --to D:\\a...\ --threads 10
  etc
  

### TESTED WITH:
* Windows 10
* python 3.9
