### INSTALLATION:
1. git clone https://github.com/DenMaslov/dz5.git
2. cd dz5
3. cd files_utility


### for using:
  1. py main.py --operation move --src D:\\b...\\ --to D:\\a...\\ --threads 10
  2. py main.py --operation copy --src D:\\b...\\ --to D:\\a...\ --threads 10
  3. etc

### Each copying/ moving call runs in thread pool

### TESTED WITH:
* Windows 10
* python 3.9
